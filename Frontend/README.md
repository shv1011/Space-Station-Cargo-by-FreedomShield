# 🚀 Space Cargo Management by Freedom Shield

A project submission for **National Space Hackathon 2025**.

## 🛰️ About the Project

**Space Cargo Management by Freedom Shield** is a futuristic solution designed to streamline and optimize the organization, allocation, and tracking of cargo in space missions. This system is tailored for maximum efficiency, safety, and scalability — ensuring successful space operations for both manned and unmanned missions.

## 🌌 Key Features

- 📦 **Smart Cargo Allocation:** Efficiently assigns cargo to available compartments based on size, weight, and priority.
- 📊 **Real-time Dashboard:** Visual representation of cargo placement using 3D models.
- 🔐 **Access Control:** Role-based user access for astronauts, mission control, and ground staff.
- ♻️ **Sustainability Tracker:** Monitors cargo types for reusability and recyclability in orbit.

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Three.js, React, TypeScript, Tailwind CSS
- **Backend:** Node.js / DJango
- **Database:** Firebase
- **3D Modeling:** Three.js or Blender models embedded in frontend
